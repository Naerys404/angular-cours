Leçon : Les Pipes en Angular

Les pipes permettent de transformer les données dans les templates Angular de manière simple et réutilisable.

1. Pipes intégrés et personnalisés

Voici les exemples de pipes intégrés et de vos pipes personnalisés :

  • Date : {{ currentDate | date:'fullDate' }}
    Original : {{ currentDate | date:'medium' }}
  • Uppercase : {{ exampleText | uppercase }}
    Original : {{ exampleText }}
  • Currency : {{ productPrice | currency:'EUR':'symbol':'1.2-2' }}
    Original : {{ productPrice }}
  • Decimal : {{ decimalValue | number:'1.2-2' }}
    Original : {{ decimalValue }}
  • JSON : 
    {{ jsonData | json }}
  • Custom Pipe (RedText):
    Original : {{ exampleText }}
  • Texte souligné :
    Original : {{ exampleText }}
  • Texte avec Jeff : {{ testText | coolJeff }}
    Original : {{ testText }}
